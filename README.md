# Steam_ModdingTool
Herramienta enfocada a linux para Modding de L4d2 y GarrysMod.

# Idea general

Herramienta similar a crowbar funcional en linux

Debe permitir al usuario poder comprimir y descompromir paquetes, cargar modelos, editar rutas de juegos y trabajar con la workshop de steam en terminos de operaciones CRUD del usuario

La conexion entre esta aplicacion y Steam sera basado en lectura de sesion

# Caracteristicas base

Gestion de paquetes .VPK y .GMA
Creacion de modelos
Descomprimir modelos
Conexion con la workshop de steam
Cargar modelos .mdl
Integracion con proton
